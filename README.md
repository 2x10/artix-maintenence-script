# artix-maintenence-script
My personal system maintenence script for artix linux

This script does some specific things such as updating using pacman, paru, flatpak, clearing cache of these package managers and also clears some caches of other places where i liked it to. 

This one also stores all the terminal outputs into a log file.

It clears the cache of GNOME apps, Discord, Steam, waybar, nwg apps, rofi, firefox and temporary user files.

Then it shows some additional about failed systemd units and a disk usage summary.
